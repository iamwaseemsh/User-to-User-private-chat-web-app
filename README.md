# User-to-User-private-chat-web-app
This is a private user to user web chat app.
Technologies used.
1. NodeJS as backend
2. MongoDB database
3. JasonWebTokens
4. EJS templates


Steps to use this web app.
i. Clone this repository
ii. write 'npm install' in terminal.
iii. write 'npm run devStart' to start app.
